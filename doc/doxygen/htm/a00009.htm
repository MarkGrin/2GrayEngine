Подробное описание

this is Function class. It is base abstract class that represent user function in engine. How to inherit and use it read in docs.

To call it you must read what arguments it need by calling attrutes function. Then you must push all the arguments. Then call payLoadFunction that is overloaded by user. And user will pop all arguments you pushed.
